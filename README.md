Jasmian Messenger 🌸

**Jasmian** — это современный мессенджер на базе C# (WPF), ориентированный на приватность и безопасность. Приложение использует архитектуру клиент-сервер на TCP-сокетах и поддерживает сквозное шифрование (End-to-End Encryption).

## ✨ Основные возможности
- **🛡 Сквозное шифрование (E2EE):** Использование гибридного алгоритма RSA + AES. Сообщения шифруются на устройстве отправителя и могут быть прочитаны только получателем. Сервер видит только зашифрованные данные.
- **🖼 Мультимедиа:** Отправка текстовых сообщений и изображений.
- **👤 Профили пользователей:** Личные кабинеты с возможностью редактирования информации (имя, телефон, био, дата рождения) и галереей фотографий профиля.
- **📝 Управление сообщениями:** Редактирование и удаление сообщений в реальном времени.
- **⚡️ Статусы:** Отображение статуса «в сети», а также индикация того, что собеседник печатает сообщение.
- **💬 Умный список чатов:** Автоматическое сохранение диалогов (даже если все сообщения удалены) и предпросмотр последнего сообщения.
- **🔍 Глобальный поиск:** Поиск пользователей по логину внутри системы.

## 🛠 Технологический стек
- **Язык:** C# (.NET)
- **Интерфейс:** WPF (XAML)
- **База данных:** MS SQL Server (Entity Framework Core)
- **Сеть:** TCP Sockets (StreamReader/StreamWriter)
- **Шифрование:** System.Security.Cryptography (RSA 2048-bit, AES 256-bit)

## 🚀 Быстрый старт

### 1. Настройка сервера (Jasmian.Server)

1.  **База данных:** Убедитесь, что у вас установлен **MS SQL Server** (Express или LocalDB).
2.  **Запуск:** При первом запуске сервер автоматически создаст базу данных `JasmianDb`. В консоли может появиться запрос на очистку БД — нажмите `Y`, если хотите начать с чистого листа.
3.  **Конфигурация:** В папке с исполняемым файлом сервера появится файл `server_config.txt`. Вы можете изменить параметры:
    - `IP=0.0.0.0` — сервер слушает все доступные сетевые интерфейсы.
    - `PORT=8888` — порт для подключения.
    - `DB_CONNECTION=...` — строка подключения к вашей базе данных SQL.

### 2. Настройка клиента (Jasmian.Client)

1.  **Подключение:** В папке с клиентом создайте (или отредактируйте созданный автоматически) файл `server_ip.txt`. 
    - Впишите туда локальный IP-адрес компьютера, на котором запущен сервер (например, `192.168.1.15`).
2.  **Шифрование:** При первом запуске и регистрации клиент автоматически генерирует пару ключей RSA:
    - `public_key.xml` — отправляется на сервер для раздачи собеседникам.
    - `private_key.xml` — **скрытый файл**, хранящийся только на вашем ПК. Никогда не передавайте его третьим лицам.
3.  **Регистрация:** Введите логин и пароль. Если сервер доступен, вы мгновенно попадете в главный интерфейс.

## 🔒 Как работает безопасность?

Мессенджер использует многоуровневую защиту:
1.  **Генерация:** Каждый пользователь имеет уникальную пару RSA-ключей.
2.  **Обмен:** При открытии чата клиенты обмениваются публичными ключами через сервер.
3.  **Шифрование:** Сообщение шифруется случайным ключом AES. Затем этот ключ AES шифруется публичным ключом RSA получателя.
4.  **Передача:** На сервер уходит зашифрованный пакет. Даже если база данных будет взломана, сообщения останутся нечитаемыми.
5.  **Дешифровка:** Получатель использует свой приватный RSA-ключ, чтобы достать ключ AES и расшифровать текст.
